# PERN Todo App

A full-stack Todo application built using the PERN stack (PostgreSQL, Express, React, Node.js).

---

## 🚀 Features

- Add new todos  
- Edit existing todos  
- Delete todos  
- Mark todos as completed  

---

## 🛠️ Tech Stack

- Frontend: React (Vite) + Tailwind CSS  
- Backend: Node.js + Express  
- Database: PostgreSQL  

---

## 📦 Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## API Endpoint

Base URL:  
http://localhost:5000
